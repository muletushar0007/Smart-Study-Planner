# 📊 Smart Study Planner

A powerful web application to help students generate custom study timetables, track productivity, and stay consistent with their study goals.

## 🚀 Key Features

- **Custom Timetable Generation**: Analyzes your exam date, syllabus completion, and daily commitments (college, sleep, distractions) to build a perfect study plan.
- **Productivity Analysis**: Calculates a productivity score and risk level based on your habits.
- **🔥 NEW: Consistency & Streak System**: 
  - **Daily Check-in**: Mark your daily goal as completed on the Dashboard.
  - **Streak Counter**: Track how many consecutive days you've stayed on track.
  - **Progress Visuals**: See your latest syllabus completion at a glance.
- **Visual Analytics**: Interactive bar charts and gauges for study vs distraction hours.

## 🛠️ Tech Stack

- **Backend**: Python, Flask
- **Database**: SQLite, SQLAlchemy
- **Frontend**: HTML5, CSS3 (Glassmorphism), JavaScript
- **Visualization**: Matplotlib

## 🏃 How to Run

1.  **Install Dependencies**: 
    ```bash
    pip install flask flask-sqlalchemy matplotlib
    ```
2.  **Start the App**:
    ```bash
    python app.py
    ```
3.  **Access in Browser**:
    Open `http://127.0.0.1:5000/`

## 💡 How to Use the Streak System

1.  **Create a Plan**: Fill out your subjects and exam details on the home page.
2.  **Visit Dashboard**: Head to the "My Dashboard" page.
3.  **Mark Completed**: Click "Mark Today Completed" once you've finished your study sessions for the day.
4.  **Stay Consistent**: Don't miss a day to keep your 🔥 streak alive!
